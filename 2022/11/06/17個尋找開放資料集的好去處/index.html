17個尋找開放資料集的好去處

發表於 2022-11-06 更新於 2022-11-07 分類於 Data Science 閱讀次數：
你曾經有過不知去哪裡尋找參考用資料集的困擾嗎？從事資料科學領域的工作者，最需要的就是可靠且易使用的資料。在這邊筆者分享17個自己覺得易使用的資料來源，供同業參考切磋。

你曾經有過不知去哪裡尋找參考用資料集的困擾嗎？從事資料科學領域的工作者，最需要的就是可靠且易使用的資料。在這邊筆者分享17個自己覺得易使用的資料來源，供同業參考切磋。

Google Dataset Search
谷歌的強大不用解釋，這項搜尋工具可供使用者針對特定議題搜尋可用的公開數據，前提是你非常清楚自己需要什麼數據。

data.world
data.world是一個專為分享開放數據而設計的社群平台，模式類似於Github。他的搜尋介面使用起來很便利，且有提供csv、json等格式的預覽。除了可透過瀏覽器使用外，該平台提供了一個名為〝datadotworld〞的Python Library，讓使用者可以在程式碼中呼叫RESTful API並直接取用數據。

Kaggle
全球最大的資料科學社群，隨時都有資料分析競賽在平台上進行，也有許多使用者會在此分享資料集與程式碼。
除了競賽與資料集外，Kaggle也提供了原生的雲端IDE，讓使用者可以直接在平台上以GPU進行分析與訓練，對電腦算力不足的使用者來說是一大福音。

Papers With Code
AI領域論文及程式碼搜尋神器，該平台將ArXiv上最新機器學習的論文與Github中的程式碼對應起來，並依照SOTA、Datasets、Methods進行分類，讓使用者可以快速地找到對應的論文、程式碼與資料集。
除了尋找論文與資料集外，平台還能顯示該則論文程式碼的流行度、Github上程式碼的收藏數等，能協助使用者判讀目前SOTA的趨勢。

UCI Machine Learning Repository
由加州大學爾灣分校的David Aha於1987年創建，是最老牌的機器學習開放資料集分享平台，上面有許多機器學習領域裡面最經典的資料集。

Datahub
由數據服務商Datopian提供的開放資料平台，依照內容類別提供大量政府、財經、商業娛樂、教育、環境等方面之數據。

政府資料開放平臺（data.gov.tw）
台灣政府提供的資料開放平台，內容以台灣政府機關單位資料為主。資料集以內容進行分類，適合用於研究台灣相關議題。

Data.gov
美國政府提供的資料開放平台，使用上以站內搜尋為主，預設類別較少，僅有農業、氣候、能源、地方政府、海運、海洋、長照等。

data.europa.eu
歐盟政府提供的資料開放平台，分類詳細且易於搜尋，除了可直接下載資料集外，同時提供了SPARQL Search功能。

Hugging Face
Hugging Face原本以提供chatbot服務為主，並在github上開放了一個Transfomer模型庫，沒想到這個庫火紅了起來。
目前提供了超過100,000個預訓練模型及10,000個以上的資料集，並撰寫了一個Python Library供使用者直接在程式中呼叫模型與資料集，非常方便。

Awesome Datasets
這是一個Github上的開源資料分享專案，這個專案的Contibuter們分享了各式各樣類別的資料來源，包括許多非常專門（冷門）的資料源。

Tidy Tuesday
同樣是Github上的開源資料分享專案，創建目的是讓Contibuter們定期性地分享R語言的資料集，作為學習與訓練的用途。

Yahoo Finance
最老也是最經典的財經媒體與資料庫，是許多股票數據分析的原始資料源，提供API讓使用者直接抓取資料，但使用前須先申請帳號以及API Token。

Python API Wrappers
這是一個Github上的專案，目的是整理羅列所有開放資料集的API Wrappers，以便於讓使用者快速找到你的程式碼所需要的API。

Pandas Data Reader
由Pandas社群所開發的Python 函式庫，讓使用者可直接使用Pandas匯入內建的資料集，資料集來源主要是財經類資料庫，包括Yahoo Finance、Quandl等等。

TensorFlow Datasets
TensorFlow內建的資料集，依照機器學習領域進行分類，如Object Detection、Recommendation等，對使用者練習TensorFlow來說非常方便。

Imagenet
ImageNet是一個大型視覺資料庫，用於視覺目標辨識軟體研究。該專案已手動標註了1500萬以上的圖像，並含有2萬個以上的典型類別。

除了以上17個資料來源之外，也有一些來源擁有高品質的資料，但在使用上不是那麼便利的，這邊筆者一併提供：

Data Is Plural
這個以分享資料集為主的電子報是由BuzzFeed News的編輯Jeremy Singer-Vine所發起，標榜每周分享一次新的開源資料集，資料源同時會整理在一張Google Sheet中。缺點是資料集並沒有分類，要從這裡找到需要的資料集並不容易。

Reddit Datasets
Reddit的Datasets討論版中有許多版友分享開源資料集，若有需要也可以上板發文徵求資料集，但爬文找資料集的效率並不高。
